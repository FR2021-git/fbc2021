== Description ==

The CMSWP upgrade plugin adds the functionality to upgrade the plugins cmswp provides.

== Installation ==

1. Copy the `cmswp-upgrade` directory into your `wp-content/plugins` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. That's it! :)
